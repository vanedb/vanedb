# vanedb (Rust) C API Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a `vanedb-capi` workspace crate exposing the VaneDB Rust core through `#[no_mangle] extern "C"` functions (symbol prefix `vanedb_rs_`), built as a `staticlib`+`cdylib`, with a cbindgen-generated header, mirroring the same C-ABI contract the C++ side implements.

**Architecture:** New crate `vanedb-capi` in the Cargo workspace, depending on `vanedb` (with `mmap`). `src/lib.rs` wraps each core type behind raw-pointer handles (`Box::into_raw`/`from_raw`), converting C pointers+lengths to slices via `std::slice::from_raw_parts` and `Result::Err` into error returns. `build.rs` + cbindgen emit `include/vanedb_rs_capi.h`. Rust integration tests exercise the `extern "C"` functions for behavior.

**Tech Stack:** Rust 2021, `vanedb` (path dep), `cbindgen` (build-dep), cargo test.

**Spec:** `docs/superpowers/specs/2026-05-28-vanedb-bench-design.md` (in `vanedb/vanedb-bench`), §5 C-ABI Contract. Mirror of `vanedb-cpp`'s C API (`vanedb_cpp_*` ↔ `vanedb_rs_*`).

## File Structure

| File | Responsibility |
|------|----------------|
| `Cargo.toml` (workspace root) | Add `vanedb-capi` to `members` |
| `vanedb-capi/Cargo.toml` | Crate manifest: `crate-type = ["staticlib","cdylib","rlib"]`, dep on `vanedb`, build-dep `cbindgen` |
| `vanedb-capi/src/lib.rs` | All `vanedb_rs_*` `extern "C"` functions + `to_metric` helper |
| `vanedb-capi/build.rs` | cbindgen invocation generating the header |
| `vanedb-capi/cbindgen.toml` | cbindgen config (C output, `vanedb_rs_` filter) |
| `vanedb-capi/tests/capi.rs` | Rust integration tests calling the `extern "C"` fns |

**Conventions (every wrapper):**
- Handles: `Box::into_raw(Box::new(obj))` returns `*mut T`; `_free` does `drop(Box::from_raw(ptr))` guarded by `is_null()`.
- Inputs: `slice::from_raw_parts(ptr, len)`; C strings via `CStr::from_ptr(path).to_str()`.
- `Result::Ok` → success (`0` for `int`, the object pointer, or count); `Result::Err` → error (`1`, null, or `0` count).
- Metric passed as `u32` (C enum is an int): `0=L2, 1=Cosine, 2=Dot`.
- Functions are `pub unsafe extern "C"` (they dereference raw pointers). Inputs are valid by contract (the bench controls them).

pub unsafe extern "C" fn vanedb_rs_store_new(dim: usize, metric: u32) -> *mut VectorStore {

#[no_mangle]
pub unsafe extern "C" fn vanedb_rs_store_add(s: *mut VectorStore, id: u64, v: *const f32) -> i32 {
    let store = &*s;
    let vec = slice::from_raw_parts(v, store.dimension());
    match store.add(id, vec) { Ok(()) => 0, Err(_) => 1 }
}
#[no_mangle]
pub unsafe extern "C" fn vanedb_rs_store_search(
    s: *mut VectorStore, q: *const f32, k: usize, out_ids: *mut u64, out_dists: *mut f32,
) -> usize {
    let store = &*s;
    let query = slice::from_raw_parts(q, store.dimension());

                *out_dists.add(i) = r.distance;
            }
            res.len()

        Err(_) => 0,
    }
}

    if !s.is_null() { drop(Box::from_raw(s)); }
}

#[no_mangle]
pub unsafe extern "C" fn vanedb_rs_hnsw_new(
    dim: usize, metric: u32, capacity: usize, m: usize, ef_construction: usize, seed: u64,
) -> *mut HnswIndex {
    match HnswIndex::builder(dim, to_metric(metric))
        .capacity(capacity).m(m).ef_construction(ef_construction).seed(seed).build()
    {
        Ok(h) => Box::into_raw(Box::new(h)),
        Err(_) => std::ptr::null_mut(),
    }
}
#[no_mangle]
pub unsafe extern "C" fn vanedb_rs_hnsw_add(h: *mut HnswIndex, id: u64, v: *const f32) -> i32 {
    let idx = &*h;
    let vec = slice::from_raw_parts(v, idx.dimension());
    match idx.add(id, vec) { Ok(()) => 0, Err(_) => 1 }
}
#[no_mangle]
pub unsafe extern "C" fn vanedb_rs_hnsw_search(
    h: *mut HnswIndex, q: *const f32, k: usize, ef_search: usize,
    out_ids: *mut u64, out_dists: *mut f32,
) -> usize {
    let idx = &*h;
    idx.set_ef_search(ef_search);
    let query = slice::from_raw_parts(q, idx.dimension());
    match idx.search(query, k) {
        Ok(res) => {
            for (i, r) in res.iter().enumerate() {
                *out_ids.add(i) = r.id;
                *out_dists.add(i) = r.distance;
            }
            res.len()
        }
        Err(_) => 0,
    }
}
#[no_mangle]
pub unsafe extern "C" fn vanedb_rs_hnsw_save(h: *mut HnswIndex, path: *const c_char) -> i32 {
    let idx = &*h;
    match CStr::from_ptr(path).to_str() {
        Ok(p) => match idx.save(p) { Ok(()) => 0, Err(_) => 1 },
        Err(_) => 1,
    }
}
#[no_mangle]
pub unsafe extern "C" fn vanedb_rs_hnsw_load(path: *const c_char) -> *mut HnswIndex {
    match CStr::from_ptr(path).to_str() {
        Ok(p) => match HnswIndex::load(p) {
            Ok(h) => Box::into_raw(Box::new(h)),
            Err(_) => std::ptr::null_mut(),
        },
        Err(_) => std::ptr::null_mut(),
    }
}
#[no_mangle]
pub unsafe extern "C" fn vanedb_rs_hnsw_free(h: *mut HnswIndex) {
    if !h.is_null() { drop(Box::from_raw(h)); }
}

```rust
#[no_mangle]
pub unsafe extern "C" fn vanedb_rs_mmap_build(
    path: *const c_char, dim: usize, metric: u32,
    ids: *const u64, vecs: *const f32, n: usize,
) -> i32 {
    let p = match CStr::from_ptr(path).to_str() { Ok(s) => s, Err(_) => return 1 };
    let mut b = match MmapVectorStoreBuilder::new(dim, to_metric(metric)) {
        Ok(b) => b, Err(_) => return 1,
    };
    let id_slice = slice::from_raw_parts(ids, n);
    for i in 0..n {
        let v = slice::from_raw_parts(vecs.add(i * dim), dim);
        if b.add(id_slice[i], v).is_err() { return 1; }
    }
    match b.save(p) { Ok(()) => 0, Err(_) => 1 }
}
#[no_mangle]
pub unsafe extern "C" fn vanedb_rs_mmap_open(path: *const c_char) -> *mut MmapVectorStore {
    match CStr::from_ptr(path).to_str() {
        Ok(p) => match MmapVectorStore::open(p) {
            Ok(m) => Box::into_raw(Box::new(m)),
            Err(_) => std::ptr::null_mut(),
        },
        Err(_) => std::ptr::null_mut(),
    }
}
#[no_mangle]
pub unsafe extern "C" fn vanedb_rs_mmap_search(
    m: *mut MmapVectorStore, q: *const f32, k: usize, out_ids: *mut u64, out_dists: *mut f32,
) -> usize {
    let store = &*m;
    let query = slice::from_raw_parts(q, store.dimension());
    match store.search(query, k) {
        Ok(res) => {
            for (i, r) in res.iter().enumerate() {
                *out_ids.add(i) = r.id;
                *out_dists.add(i) = r.distance;
            }
            res.len()
        }
        Err(_) => 0,
    }
}
#[no_mangle]
pub unsafe extern "C" fn vanedb_rs_mmap_free(m: *mut MmapVectorStore) {
    if !m.is_null() { drop(Box::from_raw(m)); }
}
```
